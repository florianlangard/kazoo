# Kazoo :postal_horn:

Built with electronjs & forge :tada:

![image](img/kazoo.jpg)

Install :
```bash
npm install
```

Start the app :
```bash
npm start
```

build it :
```bash
npm run make
```

publish a new release with github actions :
```bash
git tag vX.X.X
git push origin vX.X.X
```

delete local tag :
```bash
git tag -d vX.X.X
```

delete remote tag :
```bash
git push origin --delete tag vX.X.X
```
